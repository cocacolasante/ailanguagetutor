# AI Language Tutor -- Dashboard, Learning System, and Feature Roadmap

## Overview

We are redesigning the application to introduce a **true dashboard**,
expanded **learning features**, **gamification mechanics**, **AI tutor
personalities**, and a **structured curriculum system**.

Users will earn **Fluency Proficiency Points (FP)** through learning
activities and conversations, which contribute toward language-specific
levels and leaderboard rankings.

------------------------------------------------------------------------

### Most Recent Conversation

Displays information about the user's last conversation.

Displayed Data: - Conversation length - Language used - Level -
AI-generated learning summary

Actions: - Continue conversation - View summary

Example: Most Recent Conversation\

Level: Intermediate\
Length: 12 minutes

Summary: Practiced ordering food and discussing travel plans.
